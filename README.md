Electric Vehicle Sales & Adoption Analysis

Project Overview:

This project analyzes Electric Vehicle (EV) sales trends and adoption patterns using data analysis and machine learning techniques. It focuses on understanding growth, seasonal trends, and factors influencing EV adoption.

 Objectives:
Analyze EV sales trends over time
Identify seasonal patterns
Perform data visualization (EDA)
Build predictive models for EV sales

Dataset:
EV sales dataset
Contains--
Date
EV sales quantity
Vehicle categories

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn

Methodology:
Data Preprocessing----Converted date column to datetime,Checked missing values,Feature Engineering,Extracted Year and Month
Exploratory Data Analysis (EDA)----Heatmaps,Pairplots,Histograms,Boxplots
Model Building----Linear Regression,KNN Regressor
Evaluation----  Model performance comparison

Visualizations:
--Sales trend over years
--Category-wise sales
--Correlation heatmap
--Monthly distribution plots

Algorithms Used:
--Linear Regression
--K-Nearest Neighbors (KNN)

Results:
--Identified increasing EV adoption trend
--Seasonal variations observed
--Models provide basic prediction capability
